## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sveld03/cs35l_project.git
   cd HOTH-server
   ```
2. **Install dependencies**
    ```bash
    npm install
    ```
3. **Run the Application**
    ```bash
    node server.js
    ```
## Tech Stack
* Node.js
* Express.js
* Javascript
* MongoDB
* jsonwebtoken (JWT auth implementation library)
